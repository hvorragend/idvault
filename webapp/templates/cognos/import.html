Cognos Berichtsübersicht hochladen

Laden Sie hier die Berichtsübersicht aus agree21Analysen hoch. Die Datei muss tab-separiert (TSV) sein und folgende Spalten enthalten: Berichtsname, Anwendung, Suchpfad, Package, Eigentümer usw.

Tipp: Der Import ist idempotent – bereits vorhandene Berichte werden aktualisiert (Metadaten), ihr Bearbeitungsstatus bleibt erhalten. Neu importierte Berichte erhalten den Status Neu.
Erlaubte Formate: .txt, .tsv, .csv — Encoding: UTF-8 oder Latin-1.

Erwartetes Dateiformat

Zeile 1 (optional): Berichtsübersicht | Anz. Berichte: … — wird übersprungen.

Zeile 2: Tab-separierte Spaltenköpfe, z.B.:

Umfeld Bank-ID Anwendung Berichtsname Suchpfad Package Eigentümer Berichtsbeschreibung Erstelldatum Änderungsdatum …

Zahlen im deutschen Format (Tausender-Punkt): 1.057 wird als 1057 interpretiert.
Datumsfelder werden als Text gespeichert (z.B. 11.09.2024).
